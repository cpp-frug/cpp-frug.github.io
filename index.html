C++ FRench User Group

Développeurs C++ de tous les pays, rencontrez-vous!

Follow me on GitHub

Le C++ FRenc User Group a pour vocation de promouvoir et d'encourager les développeurs C++ francophones à se rencontrer pour discuter et échanger sur le langage C++.

Chaque ville où se déroule un meeting C++ est invitée à partager les présentations et éventuelles vidéos réalisées à cette occasion de façon à ce que le travail fourni bénéficie au plus grand nombre.

Liste des groupes actifs:

  • Paris
  • Montpellier

Il n'y a pas encore de meeting C++ dans votre ville? Organisez-en un! Le FRUG est là pour vous aider dans cette démarche en mettant à votre disposition des outils et ressources pour vous faciliter la tâche. Vous pouvez aussi compter sur l'expérience des autres organisateurs pour vous aider à vous lancer et vous amuser un maximum!
